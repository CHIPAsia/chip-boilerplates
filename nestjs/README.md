## Notes

#### Execute the following to run the app:

1. Make sure to install all dependencies

   ```bash
   yarn                 //install deps
   ```

1. Next

   ```bash
   yarn start:dev       //start app
   ```

1. On new terminal, run
   ```
   npx tailwindcss -i ./src/input.css -o ./public/output.css --watch
   ```
